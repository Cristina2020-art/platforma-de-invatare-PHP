Transacții în PDO

<?php
try {
  $pdo->beginTransaction();

  $pdo->exec("INSERT INTO conturi (nume, sold) VALUES ('Ion', 1000)");
  $pdo->exec("UPDATE conturi SET sold = sold - 100 WHERE nume = 'Ion'");

  $pdo->commit();
} catch (Exception $e) {
  $pdo->rollBack();
  echo "Eroare: " . $e->getMessage();
}
?>

Indexare și optimizare interogări

Crearea indexurilor pentru a accelera căutările:

CREATE INDEX idx_nume ON utilizatori(nume);

Stored Procedures

Proceduri stocate pentru logică la nivelul bazei de date:

DELIMITER //
CREATE PROCEDURE getUserById(IN userId INT)
BEGIN
  SELECT * FROM utilizatori WHERE id = userId;
END //
DELIMITER ;
