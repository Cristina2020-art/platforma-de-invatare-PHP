Ce sunt excepțiile?

Excepțiile sunt erori care pot fi prinse și gestionate pentru a preveni oprirea neașteptată a programului.

Structura try-catch

<?php
try {
  // Cod care poate genera o excepție
  if (!file_exists("fisier.txt")) {
    throw new Exception("Fișierul nu există.");
  }
} catch (Exception $e) {
  echo "Eroare: " . $e->getMessage();
}
?>

Blocul finally

Codul din finally se execută întotdeauna, indiferent dacă a apărut o excepție sau nu.

<?php
try {
  // cod
} catch (Exception $e) {
  // tratare
} finally {
  // cod executat oricum
}
?>
