Clase și obiecte

O clasă este un tip de date definit de utilizator care conține proprietăți (variabile) și metode (funcții).

Exemplu de clasă cu proprietăți și metode

<?php
class Persoana {
  public $nume;
  public $varsta;

  public function __construct($nume, $varsta) {
    $this->nume = $nume;
    $this->varsta = $varsta;
  }

  public function salut() {
    return "Salut, numele meu este " . $this->nume . " și am " . $this->varsta . " ani.";
  }
}

$persoana1 = new Persoana("Ana", 25);
echo $persoana1->salut();
?>

Metode speciale

  • __construct() – constructorul clasei, se execută la crearea obiectului
