Introducere în PHPUnit

Framework standard pentru testarea unităților în PHP.

Exemplu test simplu

<?php
use PHPUnit\Framework\TestCase;

class CalculTest extends TestCase {
  public function testAdunare() {
    $rezultat = 2 + 3;
    $this->assertEquals(5, $rezultat);
  }
}
?>
