Ce este un API REST?

REST este un stil arhitectural pentru comunicarea între aplicații, folosind protocoale HTTP.

Exemplu simplu: cerere GET cu cURL în PHP

<?php
$url = "https://api.example.com/date";

$curl = curl_init($url);
curl_setopt($curl, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($curl);
curl_close($curl);

$data = json_decode($response, true);
print_r($data);
?>

Explicații:

  • curl_init() – inițializează o sesiune cURL
  • curl_setopt() – setează opțiuni, aici returnează rezultatul ca string
  • curl_exec() – execută cererea
  • json_decode() – decodează răspunsul JSON într-un array PHP
