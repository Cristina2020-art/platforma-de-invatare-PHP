Tipuri de date

PHP suportă mai multe tipuri de date, inclusiv:

  • String: șiruri de caractere
  • Integer: numere întregi
  • Float: numere reale
  • Boolean: valori adevărat sau fals
  • Array: tablouri de valori
  • Object: obiecte
  • NULL: valoare nulă

Structuri de control

Structurile de control determină fluxul execuției unui program. Exemple comune includ:

  • if: execută un bloc de cod dacă o condiție este adevărată
  • else: execută un bloc de cod dacă condiția este falsă
  • switch: selectează un bloc de cod de executat din mai multe opțiuni

Bucle

Buclele permit executarea repetată a unui bloc de cod. Tipuri comune de bucle includ:

  • for: buclă cu un număr cunoscut de iterații
  • while: buclă care se execută cât timp o condiție este adevărată
  • do...while: buclă care se execută cel puțin o dată și apoi repetă cât timp condiția este adevărată
  • foreach: buclă care iterează prin fiecare element al unui array
