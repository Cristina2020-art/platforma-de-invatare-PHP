Hashing-ul parolelor

Parolele trebuie stocate criptat, nu în clar.

Funcții PHP recomandate

  • password_hash() – creează un hash sigur
  • password_verify() – verifică parola la login

Exemplu de înregistrare

<?php
$parola = 'parola_user';
$hash = password_hash($parola, PASSWORD_DEFAULT);
echo $hash;
?>

Exemplu de verificare

<?php
$parola_introdusa = 'parola_user';
$hash_stocat = '$2y$10$...'; // hash preluat din baza de date

if (password_verify($parola_introdusa, $hash_stocat)) {
    echo "Parola corectă";
} else {
    echo "Parola greșită";
}
?>
