Funcții în PHP

Acasă | Cursuri | Modul 1

Ce este o funcție?

Funcțiile sunt blocuri de cod reutilizabile care execută o anumită sarcină. Ele pot primi parametri și pot returna valori.

Definirea unei funcții

<?php
function salut($nume) {
  return "Salut, " . $nume . "!";
}
?>

Apelarea unei funcții

<?php
echo salut("Maria");  // Afișează: Salut, Maria!
?>

Funcție fără parametri

<?php
function mesaj() {
  echo "Bun venit la cursul PHP!";
}
mesaj();
?>

Funcție cu parametri și valoare returnată

<?php
function aduna($a, $b) {
  return $a + $b;
}
$suma = aduna(5, 7);
echo "Suma este: " . $suma;
?>
