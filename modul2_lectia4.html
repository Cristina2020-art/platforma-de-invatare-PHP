Ce este un namespace?

Namespace-urile organizează codul pentru a evita conflictele între clase sau funcții cu același nume.

Exemplu simplu

<?php
namespace Proiect\Util;

class Ajutor {
  public static function salut() {
    return "Salut din namespace!";
  }
}

// Folosire:
echo \Proiect\Util\Ajutor::salut();
?>

Importarea unui namespace cu use

<?php
use Proiect\Util\Ajutor;

echo Ajutor::salut();
?>
