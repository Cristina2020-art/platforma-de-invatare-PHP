Sintaxa de bază în PHP

Acasă | Cursuri | Modul 1

1. Declarația PHP

Codul PHP începe cu <?php și se încheie cu ?>. Tot codul între aceste tag-uri este interpretat de server.

2. Comentarii

// Comentariu pe o linie
# Alt comentariu pe o linie
/*
Comentariu
pe mai multe
linii
*/

3. Variabile

Variabilele în PHP încep cu simbolul $ urmat de numele variabilei. Nu trebuie să fie declarate explicit înainte de folosire.

<?php
$mesaj = "Salut!";
$varsta = 25;
echo $mesaj;
?>

4. Tipuri de date de bază

  • string - text
  • int - număr întreg
  • float - număr zecimal
  • bool - true/false
  • array - listă de valori

5. Structuri de control

Exemplu if-else:

<?php
$nota = 8;
if ($nota >= 5) {
  echo "Ai trecut.";
} else {
  echo "Nu ai trecut.";
}
?>

6. Buclă for

<?php
for ($i = 1; $i <= 5; $i++) {
  echo "Număr: $i <br>";
}
?>
