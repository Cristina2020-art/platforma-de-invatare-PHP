Ce este PHP?

PHP este un limbaj de programare server-side folosit pentru a crea pagini web dinamice și aplicații web. Este foarte popular datorită ușurinței cu care poate fi învățat și integrat cu baze de date.

Primul script PHP

Vom crea un fișier PHP simplu care afișează un mesaj pe pagină.

<?php
echo "Salut, lume!";
?>

Explicație:

  • Codul PHP începe cu <?php și se termină cu ?>
  • Instrucțiunea echo afișează textul pe pagină.

Cum să-l testezi?

1. Instalează un server local, de ex. XAMPP sau Laragon.
2. Creează un fișier numit primul_script.php în folderul htdocs (sau www).
3. Copiază codul de mai sus în fișier și salvează.
4. Accesează http://localhost/primul_script.php în browser.
5. Vei vedea mesajul "Salut, lume!" afișat.
