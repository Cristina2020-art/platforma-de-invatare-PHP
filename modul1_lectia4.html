Variabile în PHP

Variabilele în PHP sunt spații de memorie care stochează date. Sunt denumite cu simbolul $ urmat de un nume valid (litere, cifre, _ și nu începe cu cifră).

Exemplu de variabile:

<?php
$nume = "Ion";
$varsta = 30;
$pret = 19.99;
$activ = true;
?>

Tipuri de date în PHP

  • string – text (ex: "Salut")
  • int – număr întreg (ex: 10)
  • float – număr zecimal (ex: 3.14)
  • bool – adevărat/fals (true/false)
  • array – colecție de valori
  • null – valoare nulă

Exemplu utilizare tipuri:

<?php
$mesaj = "Salut lume!";
$numar = 42;
$pret = 12.5;
$activ = false;
$lista = array("mere", "pere", "banane");
$valoareNula = null;
?>

Tipul variabilelor se poate schimba dinamic:

<?php
$variabila = "text";
$variabila = 10; // acum este număr
?>
