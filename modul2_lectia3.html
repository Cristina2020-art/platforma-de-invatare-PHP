Moștenirea (Inheritance)

O clasă poate moșteni proprietățile și metodele altei clase, pentru a reutiliza codul.

Exemplu:

<?php
class Animal {
  public function sunet() {
    return "Sunet generic";
  }
}

class Caine extends Animal {
  public function sunet() {
    return "Ham ham!";
  }
}

$caine = new Caine();
echo $caine->sunet(); // Afișează: Ham ham!
?>

Polimorfismul

Polimorfismul permite metodei din clasa derivată să modifice comportamentul metodei moștenite.

Explicație:

  • Metoda sunet() este redefinită în clasa Caine.
  • Astfel, comportamentul diferă în funcție de tipul obiectului.
