Filtrare și validare cu filtre PHP

<?php
$email = "exemplu@domeniu.com";
if (filter_var($email, FILTER_VALIDATE_EMAIL)) {
  echo "Email valid";
} else {
  echo "Email invalid";
}
?>

Validări personalizate

<?php
function valideazaParola($parola) {
  return strlen($parola) >= 8 && preg_match('/[A-Z]/', $parola);
}

if (valideazaParola("Parola123")) {
  echo "Parola este valida";
} else {
  echo "Parola nu este valida";
}
?>

Sanitizare date complexe

Exemplu de curățare a unui text pentru a preveni XSS:

<?php
$input = "<script>alert('XSS')</script>";
$safe_input = htmlspecialchars($input, ENT_QUOTES, 'UTF-8');
echo $safe_input;
?>
