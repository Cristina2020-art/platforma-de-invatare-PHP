Autentificare

Procesul de verificare a identității utilizatorului (login).

Exemplu simplu de login

<?php
session_start();
if ($_POST) {
    $user = $_POST['username'];
    $pass = $_POST['password'];

    // Exemplu fix, în realitate folosește baze de date și hashing!
    if ($user === 'admin' && $pass === 'parola123') {
        $_SESSION['user'] = $user;
        echo "Login reușit!";
    } else {
        echo "Utilizator sau parolă incorectă.";
    }
}
?>

Autorizare

Verificarea permisiunilor unui utilizator pentru a accesa anumite resurse.

Exemplu de verificare autorizare

<?php
session_start();
if (!isset($_SESSION['user'])) {
    header('Location: login.php');
    exit;
}
?>
