Validarea și sanitizarea datelor

Lecția 2 | Lecția 4 | Înapoi la Modul 4

Ce este validarea datelor?

Procesul de verificare dacă datele primite sunt corecte și în formatul așteptat.

Sanitizarea datelor

Curățarea datelor pentru a elimina caractere sau coduri periculoase.

Exemple PHP

Validare email

<?php
$email = $_POST['email'];
if (filter_var($email, FILTER_VALIDATE_EMAIL)) {
  echo "Email valid";
} else {
  echo "Email invalid";
}
?>

Sanitizare string

<?php
$input = $_POST['comentariu'];
$curat = filter_var($input, FILTER_SANITIZE_STRING);
echo $curat;
?>

Prevenirea XSS

Folosirea funcției htmlspecialchars() pentru afișarea datelor:

<?php
echo htmlspecialchars($input, ENT_QUOTES, 'UTF-8');
?>
