Instalare server local (XAMPP)

Acasă | Cursuri | Modul 1

Ce este XAMPP?

XAMPP este un pachet gratuit care instalează un server Apache, PHP și MySQL pe calculatorul tău, pentru a putea dezvolta și testa aplicații PHP local.

Pași instalare XAMPP:

  1. Accesează site-ul oficial: https://www.apachefriends.org/ro/index.html
  2. Descarcă versiunea potrivită pentru sistemul tău de operare (Windows, Linux sau Mac)
  3. Rulează fișierul descărcat și urmează instrucțiunile de instalare
  4. După instalare, deschide panoul de control XAMPP
  5. Pornește serviciile Apache și MySQL (apasă butoanele Start)
  6. Deschide browserul și accesează http://localhost – ar trebui să vezi pagina de start XAMPP

Testare server PHP:

1. Creează un fișier numit test.php în folderul htdocs (de exemplu: C:\xampp\htdocs\test.php)

2. Pune acest cod în fișier:

<?php
echo "Serverul PHP funcționează!";
?>

3. Accesează http://localhost/test.php în browser. Dacă vezi mesajul, totul este setat corect.
