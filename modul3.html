Modul 3: Funcții și Array-uri în PHP

Funcții Array-uri Superglobali

Funcții în PHP

Funcțiile sunt blocuri de cod reutilizabile care execută o anumită sarcină. Ele pot primi parametri și pot returna valori.


<?php
function salut($nume) {
    return "Salut, " . $nume . "!";
}

echo salut("Ana");
?>

Array-uri în PHP

Array-urile sunt structuri de date care pot stoca mai multe valori într-o singură variabilă.


<?php
$fructe = array("Măr", "Banana", "Cireșe");
echo $fructe[1]; // Afișează "Banana"

$persoana = array(
    "nume" => "Ion",
    "varsta" => 30,
    "oras" => "Cluj"
);
echo $persoana["oras"]; // Afișează "Cluj"
?>

Superglobali în PHP

Superglobalii sunt variabile predefinite în PHP care sunt accesibile din orice parte a scriptului. Exemple comune includ:

  • $_GET: Date transmise prin URL
  • $_POST: Date transmise prin formulare
  • $_SESSION: Date stocate în sesiune
  • $_COOKIE: Date stocate în cookie-uri

<?php
// Exemplu de utilizare a $_GET
if (isset($_GET['nume'])) {
    echo "Salut, " . htmlspecialchars($_GET['nume']) . "!";
}
?>

Pentru a testa acest script, accesează pagina cu un parametru în URL, de exemplu: modul3.html?nume=Ana
